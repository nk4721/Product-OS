# Security and privacy

- enforce tenant isolation and least privilege
- validate user and application authorization for every protected operation
- keep secrets and tokens out of browser code, logs and repository files
- minimize copied personal and commercial data
- use purpose-based access and retention rules
- protect documents and user-defined fields containing sensitive information
- audit material reads and writes where required
- align telemetry and support exports with privacy rules
- document subprocessors and external data flows per product
